# Google-Ads-Performance-Analysis

# Marketing Campaign Analysis
This project analyzes Google Ads campaign performance using Python and Excel.
The dataset contains 2600 rows with raw, uncleaned marketing data.  

Key steps: 
- Data cleaning and preprocessing;
- Metric calculation (CTR, CPC, CPA, ROAS, ROI);
- Dashboard creation in Excel.

All steps in Python:
- First initial exploration
- Renaming columns for better usability, fixing wrong datatypes and inconsistent data entries
- Missing values analysis and careful imputation
- Calculations of additional metrics.

Key features in Excel Dashboard:
- Scatterplot 'Cost vs Revenue for each campaign': checks for the most and least profitable campaigns
- Line Chart 'Total Conversions by Weekday': checks if conversions are getting higher or lower in particular days
- Table 'Main Metrics for Each Campaign' with conditional formatting: helps quickly skim through the main metrics for each campaign and spot highest and lowest values
- Pie charts 'Conversion percentage across Keywords and Devices': checks if conversions are getting higher or lower depending on the keyword or device used
- Block 'Key Insights': contains short summaries of the dashboard charts.

# Аналіз маркетингових кампаній
Цей проект присвячений аналізу ефективності кампаній Google Ads за допомогою Python та Excel.
Набір даних містить 2600 рядків із необробленими, неочищеними маркетинговими даними.  

Основні етапи: 
- Очищення та попередня обробка даних;
- Розрахунок показників (CTR, CPC, CPA, ROAS, ROI);
- Створення дашборду в Excel.

Усі етапи в Python:
- Первинне дослідження;
- Перейменування стовпців для зручності використання, виправлення неправильних типів даних та невідповідних записів;
- Аналіз відсутніх значень та їх заповнення;
- Розрахунок додаткових показників.

Що містить дашборд в Excel:
- Точковий графік «Порівняння Ціни та Доходу Кожної кампанії»: дозволяє визначити найбільш та найменш прибуткові кампанії;
- Лінійний графік «Кількість Конверсій по Днях Тижня»: дозволяє перевірити, чи кількість конверсій зростає/зменшується у певні дні;
- Таблиця «Загальні метрики для Кожної Кампанії» з умовним форматуванням: допомагає швидко переглянути основні показники для кожної кампанії та виявити найвищі та найнижчі значення;
- Кругові діаграми «Відсоток Конверсій за Ключовими Словами та Пристроями»: перевіряє, чи зростають/зменшуються конверсії залежно від використовуваного ключового слова або пристрою;
- Блок «Ключові моменти аналізу»: містить короткі підсумки, винесені діаграм.
